Firebase + Netlify · Adım adım

1. Firebase Projesi Kur

1

console.firebase.google.com aç → Add Project

İsim: yunus-site (ya da istediğin)

Google Analytics: kapatabilirsin (gerek yok)

2

Web uygulaması ekle → Sol menüde ⚙️ Project settings → Your apps → </> ikonuna tıkla

App nickname: yunus-web → Register app

Çıkan config objesini kopyala, firebase-config.js'e yapıştır:

const FIREBASE_CONFIG = {
  apiKey:            "AIzaSy...",
  authDomain:        "yunus-site.firebaseapp.com",
  projectId:         "yunus-site",
  storageBucket:     "yunus-site.appspot.com",
  messagingSenderId: "123456789",
  appId:             "1:123:web:abc"
};
3

Firestore Database oluştur → Sol menü: Build → Firestore Database

→ Create database → Start in test mode (şimdilik)

Region: eur3 (europe-west) seç → Enable

4

Firestore Rules → Rules sekmesi → şunu yapıştır:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Herkes okuyabilir (site ziyaretçileri)
    match /site/content {
      allow read: if true;
      // Sadece giriş yapmış kullanıcı yazabilir
      allow write: if request.auth != null;
    }
  }
}

→ Publish

5

Authentication etkinleştir → Sol menü: Build → Authentication

→ Get started → Email/Password → Enable → Save

→ Users sekmesi → Add user

E-posta: kendi mailin   Şifre: güçlü bir şifre

Not: Bu kullanıcı sadece admin.html'e giriş için. Şifreyi güvenli bir yerde sakla.

6

İlk içeriği Firestore'a yükle — Firebase Console → Firestore → Start collection

Collection ID: site → Document ID: content

Ya da admin.html'e giriş yap → Kaydet butonuna bas → otomatik oluşturur (daha kolay).

2. Netlify'a Deploy Et

1

Bilgisayarda bir klasör aç: yunus-site/

İndirdiğin tüm dosyaları içine koy (index.html, admin.html, content.js, firebase-config.js, projects/ klasörü)

2

Terminal aç, klasöre gir:

cd yunus-site
git init
git add .
git commit -m "bismillah, ilk commit"

github.com'da yeni repo aç: yunus-site (private olabilir)

git remote add origin https://github.com/KULLANICI_ADIN/yunus-site.git
git push -u origin main
3

netlify.com → Add new site → Import an existing project → GitHub

Repoyu seç → Build command: boş bırak → Publish directory: / → Deploy

~30 saniyede site canlıya çıkar: random-name.netlify.app

4

Custom domain → Netlify: Site settings → Domain management → Add custom domain

Namecheap'te: Domain → Manage → Nameservers → Custom DNS → Netlify'ın verdiği 2 NS'i gir

~10 dakika bekle, SSL otomatik aktif olur.

3. Günlük Kullanım

Site içeriği değiştirmek

→

siteadresin.com/admin.html aç → Firebase mailin + şifrenle giriş yap

İstediğin bölümü düzenle → Kaydet → anında Firestore'a yazılır, site güncellenir

Yeni proje eklemek

→

Admin → Projeler → + Yeni Proje

ID'yi not al (örn: yeniproje)

Projeler klasörüne projects/yeniproje.html dosyasını ekle (goktug25.html'den kopyala, sadece içindeki id değişiyor)

Git'e push at → Netlify otomatik deploy eder

Kod değişikliği yapmak

→
# Değişikliği yap, sonra:
git add .
git commit -m "değişiklik açıklaması"
git push

Netlify otomatik algılar, ~30 saniyede yeni versiyon canlıya çıkar.

4. Maliyet

Domain: ~$10-12/yıl (tek ücretli şey)

Netlify: Ücretsiz (100GB bant genişliği/ay)

Firebase Firestore: Ücretsiz (50K okuma/gün, 20K yazma/gün — kişisel site için sonsuz yeter)

Firebase Auth: Ücretsiz (10K kullanıcı/ay)

Toplam: ~$10-12/yıl
